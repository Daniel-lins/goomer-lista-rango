# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/Daniel-lins/goomer-lista-rango
```

```bash
# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
